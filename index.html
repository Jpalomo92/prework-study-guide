HTML

  • The head element contans information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS Stands for Cascading Style Sheets. It gives the visual aesthetic of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switched to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • ChromeDevTools is where you can see the console.log method in use.
  • To access ChromeDevTools, right click and select 'Inspect' or press CTRL+Shift+I.
  • The Console.log method is used to output a message to the web console by adding an argument.
  • Console - Refers to the test environment developers use to test their code.
  • .log() -Is a set of instructions used by a computer. It will output whaever is added in the parentheses.
  • Functions - are created using unique names followed by curly brackets that contain code for a task we are wanting to perform.
  • To execute a function, be sure to call it in your code!
